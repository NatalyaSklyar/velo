# Построение траектории движения велосипедиста

## модель
`model_training`  
Дообученная `yolo11l-pose` на изображениях велосипедистов — используется для детекции позы и координат ключевых точек на кадре.

## анализ видео
`analysis_git200/src/video_processing`  
Пайплайн: берётся sample‑кадр для фона/ROI, затем лёгкий motion‑trigger ищет момент появления движения в зоне, и только после срабатывания запускается тяжёлая трекинг‑сегментация (YOLO pose + Kalman) по отрезку. Результат пишется в единый CSV с сырыми и фильтрованными координатами.


## обработка данных
`analysis_git200/src/path_analysis`  
Постобработка CSV: выбор координат (raw/filt), интерполяция пропусков, сглаживание, расчёт метрик эпизодов и фильтрация по качеству. Далее формируется tidy‑датасет и визуализации траекторий: общий “bundle” и картинки по каждому эпизоду.
